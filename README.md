# AI Audio Transcription
Use AI to transcribe audio files.

![AI Audio Transcription](./public/screenshot.png)

## Running Locally

**1. Clone Repo**

```bash
git clone https://github.com/saurabhnimkande/ai-audio-transcription.git
```

**2. Install Dependencies**

```bash
npm i
```

**3. Run App**

```bash
npm run dev
```
